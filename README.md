# ai.petrzart.ru — онлайн-портфолио

Статический сайт-портфолио для AI-интегратора и автоматизатора бизнес-процессов.

## Файлы

- `index.html` — основная страница портфолио
- `policy.html` — политика обработки персональных данных
- `background.jpg` — фон первого экрана
- `Dockerfile` — сборка Docker-образа на базе `nginx:alpine`

## Деплой в Dokploy

1. Создать новый GitHub-репозиторий.
2. Загрузить файлы из этого архива в репозиторий.
3. В Dokploy открыть приложение сайта или создать новое приложение.
4. В Provider выбрать GitHub.
5. Указать репозиторий.
6. Dockerfile: `Dockerfile`.
7. Port: `80`.
8. Domain: `ai.petrzart.ru`.
9. Нажать Save → Deploy.

## Webhook формы

В `index.html` есть строка:

```js
const webhookUrl = 'ВАША_ССЫЛКА_ИЗ_N8N';
```

Когда будет готов production webhook в n8n, заменить значение на реальную ссылку.
