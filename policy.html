Политика обработки персональных данных

Настоящая политика описывает порядок обработки персональных данных пользователей сайта ai.petrzart.ru. Оператор персональных данных — ИП Зимин Петр.

1. Какие данные могут обрабатываться

При отправке формы обратной связи могут обрабатываться следующие данные:

  • имя;
  • контакт для связи: Telegram, телефон или email;
  • описание задачи, которую пользователь хочет обсудить;
  • техническая информация, необходимая для обработки обращения.

2. Цели обработки данных

Персональные данные используются только для:

  • обратной связи с пользователем;
  • обсуждения задачи по автоматизации;
  • подготовки предложения по проекту;
  • ведения истории обращений и улучшения качества коммуникации.

3. Передача данных третьим лицам

Данные не передаются третьим лицам, за исключением случаев, когда это необходимо для обработки заявки через используемые технические сервисы или требуется по закону.

4. Хранение и безопасность

Оператор принимает разумные меры для защиты данных от несанкционированного доступа, изменения, раскрытия или уничтожения.

5. Отзыв согласия
